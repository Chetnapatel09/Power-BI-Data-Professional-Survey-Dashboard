# 📊 Data Professional Survey Dashboard (Power BI)

## 📌 Project Overview
This project analyzes a **Data Professional Survey dataset** using **Microsoft Power BI** to uncover insights about salaries, programming languages, job roles, and work satisfaction among data professionals.

The dashboard helps explore trends in the data industry and understand how different roles compare in terms of salary, skills, and work-life balance.

## 📈 Key Insights
- **Average Salary by Job Title** – Comparison between Data Scientists, Data Engineers, Analysts, and Architects.
- **Country Distribution** – Survey participation by country.
- **Favorite Programming Languages** – Most preferred languages among data professionals.
- **Difficulty Entering the Data Field** – Perceived difficulty of breaking into data careers.
- **Work-Life Balance Score** – Happiness rating related to work-life balance.
- **Salary Satisfaction Score** – How satisfied professionals are with their salaries.

## 📊 Dataset
The dataset contains responses from **630 data professionals** across different countries and job roles, including information on:

- Job titles  
- Average salaries  
- Favorite programming languages  
- Work-life balance ratings  
- Difficulty entering the data field
